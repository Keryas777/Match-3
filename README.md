# Match-3 (prototype)

Jeu match-3 minimal en HTML5 Canvas (mobile friendly).

## Lancer en local
Ouvre `index.html` dans un navigateur.

> Sur iOS, le tactile marche mieux via un petit serveur local (facultatif).

## Héberger sur GitHub Pages
1. Mets ces fichiers à la racine du repo
2. GitHub → Settings → Pages
3. Source: `Deploy from a branch`
4. Branch: `main` / folder `/ (root)`
5. Sauvegarde → ton jeu est en ligne sur l’URL Pages
